# arch.layered.v2

Three-layer architecture template for formal bid documents.

This is the first Phase 1 template. It proves the target workflow:

```text
material analysis -> template contract -> frozen SVG geometry -> slot injection -> manifest
```

The template supports:

- 2 to 3 layers.
- 1 to 4 items per layer.
- Fixed 1600 x 980 canvas.
- Theme variables only for color changes; geometry must remain stable.
