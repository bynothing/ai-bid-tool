# 标书文风规范

## 一、总体文风要求

严格采用**军工/政企招投标正式书面语**，仿解放军/工信部/公安部体制内标书行文风格。语言沉稳、务实、落地、逐条响应、无虚言。

### 正确文风特征
- 陈述句为主，少用修辞
- 每句有实质信息（承诺什么、用什么技术、满足什么条款）
- 章节首段用"我方…"领起，体现主体责任
- 技术描述用客观技术语言（"采用…方案""实现…功能""支持…能力"）
- 数据、指标、参数可验证

### 错误文风特征
- AI 排比铺陈、华丽修辞
- 空洞的概括性陈述不作任何技术承诺
- "赋能""底座""抓手"等互联网黑话
- 一句接一句的"通过…实现…"空洞流水账

---

## 二、禁用词库

以下词汇在正文中出现即标记警告。> 5 处命中则阶段 4 阻断。

### 2.1 AI 排比句式标记词
- 不仅是…更是…
- 全方位、多角度、深层次
- 多维度、立体化
- 从根本上…彻底…
- 完美…极致…

### 2.2 互联网/管理咨询黑话
- 赋能
- 闭环（作动词/抽象名词时禁用，作技术术语"闭环控制"除外）
- 抓手
- 底座
- 颗粒度
- 数据中台 / 业务中台
- 降本增效
- 新质生产力
- 数字化转型升级

### 2.3 空洞通用套话
- 致力于…
- 为您保驾护航
- 行业领先
- 一流 / 国际一流
- 先进的（无具体技术指标时禁用）
- 强大的（无具体能力说明时禁用）
- 完善的（无具体措施时禁用）
- 灵活的（无具体方案时禁用）
- 高效的（无具体数据时禁用）

### 2.4 AI 高频空洞句式
- 通过…的方式，实现了…的目标
- 充分考虑了…的各种因素
- 为…奠定了坚实的基础
- 具有重要的现实意义
- 极大地提升了…
- 显著地改善了…

## 三、必用句式模板

以下句式在正文中应有显著覆盖。覆盖率 < 70% 则阶段 4 阻断。

### 3.1 承诺/响应句式
```
- 我方承诺严格遵守…
- 我方完全响应…（招标文件第X条/技术要求第X项）
- 我方接受…（条款内容）
- 满足招标文件…要求
- 确保…（交付物/指标）
```

### 3.2 技术方案句式
```
- 本项目采用…方案，实现…功能
- 系统基于…架构，支持…能力
- 参照…标准/规范，完成…开发
- 按照…（技术要求），进行…设计
```

### 3.3 交付/工期句式
```
- 于合同签订后…日内完成…
- 按期交付…，确保…
- 分…个阶段实施，每阶段输出…
- 提交…（文档/代码/报告），经甲方审核确认
```

### 3.4 质量/保障句式
```
- 建立…机制，保障…
- 通过…测试/检测，验证…指标
- 提供…年/月质保服务，响应时间≤…小时
```

## 四、章节撰写规范

### 4.1 每章节必含元素
1. **开头段**：阐明本章主题 + 对标招标条款 + 我方总体承诺
2. **技术展开**：分点或分段详述方案内容，每点有实质
3. **结尾句**：对本节覆盖的招标要求做出"无偏离"声明

### 4.2 段落字数
- 每自然段 150-400 字
- 技术分点条目每点 50-150 字
- 禁止单句成段（显得空）

### 4.3 标题规范
- 一级标题：`# 第X章 标题`（中文数字）
- 二级标题：`## X.X 标题`（阿拉伯数字）
- 三级标题：`### X.X.X 标题`
- 标题不单独成章（必须有正文内容）

### 4.4 插图占位格式
```
【插图 TU-001：系统总体架构图】
位置：第X章 X.X节 第一段之后
```
正文中不出现插图占位符，所有插图位置记录在独立占位清单中。

## 五、文风自检清单

撰写完成每章后，逐条检查：

1. [ ] 是否有禁用词？
2. [ ] 是否使用了必用句式？
3. [ ] 每段是否有实质信息？
4. [ ] 是否标注了对标条款编号？
5. [ ] 是否有数据/指标支撑？
6. [ ] 是否避免了"先进的/强大的/完善的/灵活的/高效的"等空洞形容词？
7. [ ] 段落长度是否在 150-400 字范围内？
8. [ ] 是否有 AI 排比句式？

---

## 六、对标条款标注规范

正文中应标出响应了哪条招标要求。格式：

```markdown
> **【对标条款 RQ-001】** 满足招标文件技术要求 §2.1 第1条"完成所有分模块代码整合、统一入口"
```

自检时需确认：
1. 每章正文中至少出现 2 处对标标注
2. 所有强制条款（★）必须有对应标注
3. 无对标标注的段落视为"未响应任何条款"，应删除或补充标注
