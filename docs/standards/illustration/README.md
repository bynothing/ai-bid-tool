# Illustration Standards

本目录存放配图内容标准、Draw.io 集成规范和后续质量标准。

## 文档索引

| 文档 | 用途 |
| --- | --- |
| `drawing-content-standard.md` | 节点文字、箭头标签、复杂度、配色和布局约束 |
| `drawio-integration.md` | Draw.io 可编辑源文件、CLI 导出和 lint 规则 |

## 适用范围

- `src/bid_tool/illustration_v2/renderers/drawio.py`
- `src/bid_tool/illustration_v2/renderers/template_svg.py`
- `src/bid_tool/illustration_v2/templates/`
- S5 配图任务生成与审查
