# Project Docs

本目录存放项目级文档。源码目录 `src/` 只承载运行时代码、Schema、Prompt、模板契约和必要的包内资源；长期背景、架构、计划、标准和交接信息统一放在 `docs/`。

## 目录

| 路径 | 作用 |
| --- | --- |
| `docs/PROJECT_STRUCTURE.md` | 项目文件结构与放置规则 |
| `docs/workspace/` | 项目记忆系统：状态、路线图、决策、交接 |
| `docs/architecture/` | 架构设计、重构方案、系统工程计划 |
| `docs/standards/` | 内容标准、绘图标准、质量标准 |

## 新会话入口

先读根目录：

```text
WORKSPACE.md
```

再按需进入本目录：

```text
docs/workspace/STATE.md
docs/workspace/ROADMAP.md
docs/workspace/KNOWLEDGE_MAP.md
docs/PROJECT_STRUCTURE.md
```
